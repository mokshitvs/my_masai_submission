## How to Build the Dashboard in Looker Studio

### Data Sources Required
Upload these files from 01_data/processed/ to Google Sheets or BigQuery:
- cleaned_transactions.csv        (primary transactions table)
- daily_summary.csv               (trend / time-series chart)
- payment_method_breakdown.csv    (payment method pie/bar chart)
- region_breakdown.csv            (regional performance chart)
- merchant_performance_summary.csv (merchant detail table)

### Dashboard Structure

#### Page 1 – Executive Overview
Headline KPIs (Scorecard widgets):
  • Total GMV – SUM(amount_usd) from cleaned_transactions
  • Confirmed GMV – SUM(amount_usd) WHERE status = 'captured'
  • Amount at Risk – SUM(amount_usd) WHERE high_risk_flag = 1
  • Success Rate – COUNT(captured) / COUNT(all) as percentage

Trend Chart:
  • Line chart: daily_summary.csv → X: transaction_date, Y: total_gmv_usd + captured_gmv_usd

Filters:
  • Date range filter on transaction_date
  • Dropdown filter on gateway_region (APAC / EU / US)
  • Dropdown filter on payment_method

#### Page 2 – Breakdown Views
Payment Method Breakdown:
  • Pie / Donut chart: payment_method_breakdown.csv → dimension: payment_method, metric: total_gmv_usd

Region Breakdown:
  • Bar chart: region_breakdown.csv → dimension: gateway_region, metric: captured_gmv_usd

Merchant Comparison:
  • Grouped bar: merchant_performance_summary.csv → merchant_name vs captured_gmv_usd and chargeback_count

#### Page 3 – Merchant Detail Table
  • Table: merchant_performance_summary.csv with columns:
    merchant_name, merchant_category, total_transactions, total_gmv_usd,
    captured_gmv_usd, avg_risk_score, chargeback_count, chargeback_ratio_pct,
    high_risk_count
  • Filter: merchant_name
